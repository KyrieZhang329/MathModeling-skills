# Contributing

This repository is a lightweight skills scaffold for mathematical modeling contests.

## Expectations

- Keep contributions small, clear, and easy to review.
- Prefer placeholder-first additions before detailed expansion.
- Write repository files in concise English by default.

## Scope

- Add or refine skills, docs, and templates that support the contest workflow.
- Do not add heavy tooling, dependencies, CI, or build systems unless explicitly needed.
- Do not add algorithms or fabricated benchmark content to placeholder files.

## Style

- Keep new skills structured and easy to hand off.
- Keep docs and templates brief until requirements are stable.
- Preserve naming consistency across `.claude` and `.codex`.
